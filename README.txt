Displays the REST response and route information on posts, terms, users and comment pages.

== Description ==

Originally inspired by the [Meta Inspector](https://github.com/alleyinteractive/meta-inspector)
plugin, REST Inspector provides developers a couple tools to assist WordPress development when
leveraging the REST API.

In addition to the core plugin page, which displays all registered REST routes in line with
their respective args and callbacks, the power of the REST Inspector can be found inside
the Meta Box it adds to the following content types.

 * Posts
 * Terms
 * Comments
 * Users

 The plugin should work with all custom taxonomy/terms as well as any custom post types that
 exist on the install.

== Installation ==

1. Upload plugin files to `/wp-content/plugins/rest-inspector/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
